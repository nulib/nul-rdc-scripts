📖 README

What is Tessera?

A context-based WebVTT caption editor designed for archival audio with complex metadata. Perfect for cassette digitization, classical music recordings, lectures, and performances with multiple movements or speakers.

Features

  • Context-based workflow: Set piece info once, reuse for all movements
  • Full metadata in every caption for standalone clarity
  • Visual grouping by piece in timeline
  • Override performers/date for specific movements
  • Session save/load to resume work anytime
  • Multiple accessibility themes
  • Precision audio controls with keyboard shortcuts

Workflow

  1. Click "Edit Piece Context" to set title, performers, date
  2. Add movements with just name + timestamps
  3. Context automatically applies to each caption
  4. Export clean, consistent WebVTT files

Keyboard Shortcuts

  • Space/K - Play/Pause
  • ←/→ - Skip 1 second
  • Shift+←/→ - Skip 0.1 second (fine)
  • I/O - Mark Start/End times
  • Ctrl+S - Save session
  • Ctrl+E - Export VTT

Output Format

Each caption includes: Title. Movement. Performers. Conductor. Date

Example: Sonata No. 1 for Violin & Piano (1938). Allegro con brio. Composer, Violin. Frances Newman, Piano. 12/10/45
