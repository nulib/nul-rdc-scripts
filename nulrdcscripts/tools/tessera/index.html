Memory Protection & Large Files

Tessera is designed to work with preservation-quality audio files, which can be very large (often 2GB+ for uncompressed WAV files). To protect your work and system performance, Tessera includes automatic memory management.

Understanding the Protections

Large File Warning (>2GB)

When you try to load a file larger than 2GB, you'll see a confirmation dialog. Click OK to proceed or Cancel to choose a different file. This is just a warning - you can still work with the file, but be aware it may slow down your system.

3-Hour Session Timeout

Tessera automatically monitors:

  • Total session time: How long you've had the application open
  • Total audio loaded: Combined size of all audio files loaded this session

When EITHER session has been open for 3+ hours OR you've loaded 3+ GB of total audio files, Tessera will trigger its memory protection system.

What Happens During Memory Protection

Step 1: Automatic Save

Before showing you any warning, Tessera automatically saves your work with a timestamped filename. Your work is safe!

Step 2: Warning Dialog

You have two choices:

  • Click OK: Application closes immediately (recommended)
  • Click Cancel: You get 2 more minutes to finish up. Application closes automatically after 2 minutes.

Resuming Your Work

  1. Reopen Tessera
  2. Click "Load Session" or press Ctrl+O
  3. Select your timestamped autosave file
  4. All your captions and context will be restored
  5. Reload your audio file (audio files are not saved in sessions)

Memory Tips

  • Save frequently using Ctrl/Cmd+S
  • Close and reopen every 2-3 hours for best performance
  • Work on one file at a time when possible
  • If Tessera starts feeling sluggish, save and restart

Getting Started

Basic Workflow

  1. Load audio file → Click the upload area or drag & drop
  2. Set piece context → Click "Edit Piece Context"
  3. Add captions → Mark timestamps for each movement (or entire piece)
  4. Export WebVTT → Save your completed caption file

Setting Piece Context

Before adding captions, you must set the piece context. This information will be included in every caption.

Required Fields:

  • Title: Full title of the work
  • Performers: Primary performing ensemble

Optional Fields:

  • Date: Performance date
  • Conductor: Conductor name
  • Composer: Composer name
  • Additional Info: Any other relevant context

Adding Captions

  1. Play the audio using the built-in player controls
  2. Note the start time when a movement begins (or when the piece starts)
  3. Fill in: Movement Name (optional), Start Time, End Time
  4. Add Notes (optional, for your reference only)
  5. Click "Add Caption"

Time Format: HH:MM:SS.mmm (e.g., 00:03:45.250 = 3 minutes, 45.25 seconds)

Movement Name: Leave blank for entire pieces, introductions, or spoken content. The timestamps will apply to the whole piece/section.

Using Override Fields

Sometimes you need to override the piece context for specific movements. Common scenarios:

  • Multiple pieces on one recording with different performers
  • Guest artists joining for one movement
  • Movements recorded on different dates

Click "Override Details" in the Add Caption section to override Performers, Date, or Conductor for individual captions.

Editing Captions

Click the ✏️ button on any caption to edit it. A modal will open showing all the caption's data, including its original context. You can modify any field, and changes won't affect your current piece context.

Starting a New Piece

Click the "🆕 New" button next to "Edit Piece Context" to clear all captions and context and start fresh. Make sure to save your current session first!

Keyboard Shortcuts

Audio Playback:

  • Space/K - Play/Pause
  • ←/→ - Skip 1 second
  • Shift+←/→ - Skip 0.1 second (fine-tuning)
  • Ctrl+←/→ - Skip 5 seconds
  • ,/. - Frame-by-frame (0.033s)
  • J/L - Jump 10 seconds
  • ↑/↓ - Volume up/down 10%
  • M - Mute/Unmute
  • I - Mark Start Time
  • O - Mark End Time

Application:

  • Ctrl/Cmd+S - Save session
  • Ctrl/Cmd+O - Load session
  • Ctrl/Cmd+E - Export WebVTT
  • Ctrl/Cmd+N - New Context
  • ESC - Close modals

Managing Your Work

Saving Sessions:

Click "Save Session" or press Ctrl/Cmd+S. Each save creates a timestamped file (e.g., "Symphony_No_5_2024-03-15T14-30-45.json") so you never accidentally overwrite previous work. The first time you save, choose a location (like Documents/Tessera_Sessions/). Your browser will remember this location for future saves.

Loading Sessions:

Click "Load Session" or press Ctrl/Cmd+O and select your .json file. All captions and context will be restored. Don't forget to reload your audio file!

Exporting WebVTT:

Click "Export VTT" or press Ctrl/Cmd+E to save your completed caption file with a timestamped filename (e.g., "Symphony_No_5_2024-03-15T14-30-45.vtt").

Tips & Best Practices

  • Use full, formal names for titles and performers
  • Include movement markings in original language
  • Be consistent with date formatting
  • Listen through first to note approximate timestamps
  • Work chronologically from beginning to end
  • Save frequently (every 10-15 minutes)
  • Review the Preview tab before exporting
  • Timestamped filenames prevent overwrites - save as often as you like!

Troubleshooting

Audio won't load:

  • Check file format (MP3, WAV, OGG, M4A supported)
  • For files >2GB, confirm you want to proceed
  • Check available system memory

Application closes unexpectedly:

This is the memory protection system! Your work was automatically saved with a timestamp. Simply reopen Tessera, press Ctrl+O, and load your timestamped autosave file.

Can't add caption:

  • "Please set piece context first" - Click "Edit Piece Context" and fill in Title and Performers
  • "Invalid time format" - Use HH:MM:SS.mmm format
  • "End time must be after start time" - Check your timestamps

Output Format

With movement name: Title. Movement. Performers. Conductor. Date

Example: Symphony No. 5 in C minor, Op. 67. I. Allegro con brio. Chicago Symphony Orchestra. Riccardo Muti, Conductor. March 15, 2024

Without movement name (entire piece): Title. Performers. Conductor. Date

Example: Remarks by Director James Anderson. James Anderson
