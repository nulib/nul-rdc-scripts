#Tessera

A desktop application for creating WebVTT captions for audio and video content, designed for both musical performances and speaking events.

## Features

- 🎵 Support for large preservation-quality WAV files
- 📝 Detailed metadata tracking (global and per-caption)
- 👥 Speaker/performer management with roles/instruments
- 🎨 Multiple accessibility themes (including colorblind and pastel modes)
- 💾 Automatic session saving with smart memory management
- ⌨️ Keyboard shortcuts for efficient workflow
- 🎭 Support for academic lectures, concerts, panels, and more

## Installation

### Mac
1. Download `Tessera.dmg`
2. Open the DMG file
3. Drag the app to your Applications folder
4. Right-click → Open (first time only)

### Windows
1. Download `Tessera.exe`
2. Run the installer
3. Follow the installation wizard

## Quick Start

1. **Load Audio**: Click the audio upload area or press Ctrl/Cmd+O
2. **Add Captions**: Press M to add a caption at the current time
3. **Fill Metadata**: Add speakers, hosts, conductors as needed
4. **Save Session**: Press Ctrl/Cmd+S to save your work
5. **Export**: Press Ctrl/Cmd+E to export the WebVTT file

## Keyboard Shortcuts

- **Space** - Play/Pause audio
- **M** - Add caption at current time
- **←/→** - Skip backward/forward 5 seconds
- **Shift+←/→** - Skip backward/forward 1 second
- **Ctrl/Cmd+S** - Save session
- **Ctrl/Cmd+L** - Load session
- **Ctrl/Cmd+E** - Export WebVTT file

## Working with Large Files

The app is optimized for preservation-quality WAV files:
- Handles files up to 2GB comfortably
- Automatic memory management
- Smart session saving
- See BUILD_README.md for detailed guidance
